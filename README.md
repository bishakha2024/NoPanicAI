# ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/NoPanicAI.git
```

2. Open the project in Android Studio.

3. Add your Firebase `google-services.json` file.

4. Add your Gemini API key inside:

```kotlin
const val GEMINI_API_KEY = "YOUR_API_KEY"
```

5. Sync Gradle and run the application.

---

# ⚠️ Important

This repository does not include:

- API Keys
- google-services.json
- local.properties

Create your own Firebase project and Gemini API key before running the app.
